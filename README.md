# Task Management API

## 🚀 Features
- ASP.NET Core Web API
- Clean Architecture
- Entity Framework Core
- JWT Authentication
- Role-Based Authorization
- FluentValidation
- Serilog Logging
- Global Exception Middleware

## 🛠 Tech Stack
- .NET 9
- SQL Server
- EF Core
- Swagger

## 🔐 Authentication
JWT-based authentication with role support (Admin/User)

## 📦 How to Run
1. Clone repo
2. Update connection string
3. Run migrations
4. Run API

## 📸 API Testing
Use Swagger UI
